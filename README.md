# Gemini

Grab another CoreMedia article to use dynamically in your story.

Install

```bash
npm install @abcnews/gemini
```

Import

```javascript
import * as gemini from "gemini";
```

or

```javascript
const gemini = require("@abcnews/gemini");
```

Run

```javascript
gemini.fullReplace(callbackAfterReplace);
```
